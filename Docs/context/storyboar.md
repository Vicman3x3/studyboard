tenemos la idea de una plataforma estudiantil que permita al alumno poder gestionar su horario escolar, por semestre podremos ingresar las materias y a cada materia podemos registrar tareas, queremos hacer un sistema estilo jira o notion utilizano las tradicional pententes, por hacer y hechas, las tareas se podrán agregar a el calendario de google

queremos que el sistema nos arroje alertas cuando una tarea este por cumplirse y no se haya registrado coom finalizada

que por materia podamos agregar los criterios de evaluación y con ello nos ayude a hacer el conteo para saber que calificación minima necesitamos para poder aprobar, tambien quisieramos un sistema de conteo de créditos porque la calificación es ponderada

podremos igual registrar las fechas de examenes y todo esto igual a nuestro calendario de google

las materias se separan por color

podremos crear un temario para cada materia por parcial

por parcial seria bueno tener la opción de cargar docuemntación o fotos, igual en las tareas podamos cargar fotos 

la pagina de inicio tendrá un dashboard y ahi tendremos la pregunta ¿que quieres hacer hoy?

registrar tareas
mostrar avances
ver horario de clases
ver temario por materia 
cargar docuemntos por materia
